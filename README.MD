#  Restaurant Chatbot

## Overview
The Restaurant Chatbot is a conversational assistant designed to provide quick and accurate information to restaurant customers. It helps users access menu details, pricing, availability, offers, and restaurant timings through a simple chat interface, improving customer experience and reducing manual workload.

## Key Features
- Instant responses to customer queries  
- Menu and pricing information  
- Restaurant timings and basic details  
- Simple and user-friendly interaction  
- Scalable and easy to enhance  

## Technologies Used
- Python  
- Flask / FastAPI  
- RAG

## Working Principle
User inputs are processed using predefined logic or intent matching, and appropriate responses are generated in real time based on available restaurant data.
